# 版本日志

# 0.0.1 (2019-10-20)

  • 适配lin-cms-vue 0.2.1版本

# 0.0.1 (2019-08-18)

  • 权限类库新增Scan类，位于根目录下的application\lib\auth，该类用于实现自动扫描并生成控制层下控制器的命名空间数组，支持无限级子目录
  • 借助Scan类，替换原来需要在authMap类中手动配置控制器类命名空间来实现权限注解扫描的操作方式。

# 0.0.1 (2019-06-12)

  • 新增文件上传接口

  • 新增头像更新接口

ps:旧版本更新需再执行一次数据库迁移脚本命令php think migrate:run,用于新增lin_file表和lin_user表新增avatar字段

# 0.0.1 (2019-05-21)

  • 注解验证器实现

# 0.0.1 (2019-05-19)

  • 改变行为日志记录实现方式为Hook

# 0.0.1（2019-05-15）
